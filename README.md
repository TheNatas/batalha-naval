# batalha-naval
Este projeto foi desenvolvido durante o módulo inicial de Java do curso Web FullStack da Let's Code.

# Requisitos
JDK 11+

# Instruções
O jogo é semelhante ao jogo original de batalha naval com a única diferença de não possuir outros tipos de embarcações além do simples navio. Joga-se contra a máquina, tendo a opção de posicionar os próprios navios automaticamente. Vence quem acertar todos os navios adversários primeiro.

# Legenda
Para manter o jogo impresso no console do terminal, há uma lista de símbolos para representar cada informação visível ao jogador. Segue:

### Visíveis desde o início
- N: Navio do jogador
- *: Tiro certeiro
- -: Tiro errado
- X: Tiro certeiro e navio do jogador
- n: Tiro errado e navio do jogador

### Visíveis no fim do jogo
- C: Navio do computador
- 2: Navios do jogador e do computador
